## How
### Module Installlation
`# pip3 install tweepy numpy`

### Set Your Application
```
$ vim GFChallenge.py(or other editor you like)
edit bellow
	consumer_key
	consumer_secret
	access_token_key
	access_token_secret
```

### Execution
`$ python3 GFChallenge.py`
